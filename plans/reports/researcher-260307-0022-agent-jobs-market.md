| Platform | Reward Range | Notes |

**Highest single payout on record:** $3M (reported Immunefi, "ily2" handle).

AI agents running automated vulnerability scanners (fuzzing, static analysis, formal verification) are the highest-earning autonomous agents in terms of per-task payout. A single critical find pays more than months of other work.

**ClawQuest relevance:** Security audit quests (FCFS or Leaderboard by severity score) are the highest-value quest type for sponsors.

### Tier 4: $1–$30 per task — Micro-task Marketplaces (Challenging Economics)

**What:** Small bounties, content tasks, data tasks.

| Platform | Price Range | Notes |
|---|---|---|
| ClawTasks (clawtasks.com) | 1.5–20 USDC | Agent-to-agent, Base L2; agents stake 10% as guarantee |
| toku.agency (micro) | $5 per task | 85% agent cut; code review service example |
| Content writing | varies | 500-word story = 1.5 USDC |

**Problem:** Gas fee economics. A $3 bounty costs $4 in gas to claim + submit on mainnet. RoseProtocol agent reported -$8.30 net loss after 4 days of bounty hunting. L2 deployment (Base) partially mitigates this.

**ClawQuest advantage:** Escrow-on-chain model pre-funds rewards — agents don't pay gas to claim, only to receive. This directly addresses the core economic problem.

## 6. ClawQuest Strategic Implications

### Quest Types That Will Attract Most Agents + Sponsors

| Quest Type | Sponsor Appeal | Agent Appeal | Recommended Format |
|---|---|---|---|
| Security audit / vulnerability find | High ($100K+ at stake) | High (life-changing payout) | Leaderboard by severity |
| Coding bounties (fix GitHub issues) | Medium (clear ROI) | High (demonstrable skill) | FCFS or Leaderboard |
| Agent tooling / MCP integrations | High (ecosystem value) | High ($3K–$15K) | FCFS |
| Benchmark & eval creation | Medium-High | Medium | Leaderboard |
| DeFi strategy (P&L ranked) | High (protocol TVL) | High (upside) | Leaderboard |
| Content / social tasks | Low-Medium | Low (commoditized) | Lucky Draw |
| Data labeling / RLHF | Medium | Low (low pay) | Lucky Draw |

### Key Insight for Quest Design

**The economics problem is real:** Agents avoid quests where gas fees eat profits. ClawQuest's escrow model (reward pre-funded, agent receives net of gas) is a genuine structural advantage over current platforms like ClawTasks and RoseProtocol's experience.

**The verification problem is the hard part:** The highest-paying tasks (security audits, code fixes) need objective success criteria. Quests with "tests pass / code merged" or "CVE confirmed" criteria will dominate adoption.

**Pricing anchors:**
- Micro tasks: 1–20 USDC (commodity, high volume)
- Standard tasks: $50–$500 (sweet spot for most agents)
- Premium tasks: $2,500–$15,000 (serious bounties, serious agents)
- Elite tasks: $100K+ (security criticals — Immunefi tier)
